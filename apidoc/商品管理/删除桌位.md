**简要描述：删除桌位** 

**请求URL：** 
` http://127.0.0.1:9999/commodity/deleteSeat?seatId=2&shopId=6 `

**请求方式：**
- GET

**参数：** 

| 参数名 | 必选 | 类型 | 说明 |
| :----: | :----: | :----: |  :----: |
| shopId | 是 | int |商家id|
| seatId | 是 | int |桌位id|



 **返回参数说明** 
 
|参数名|类型|说明|
|:-----  |:-----|----- |
|data| int|更新数据库成功数量|
|msg|string|正确返回，这里为空。如果报错，这里是错误信息|
|ret|int|状态码 0:正常  -1:异常|


 **返回示例**
 ``` 
{
    "data": 0,
    "msg": "",
    "ret": 0,
    "count": 0
}
```
