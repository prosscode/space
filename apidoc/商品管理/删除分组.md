**简要描述：删除商家分组** 

**请求URL：** 
` http://127.0.0.1:9999/commodity/deleteGoodType?shopId=6&typeName=一层吧台 `

**请求方式：**
- GET

**参数：** 

| 参数名 | 必选 | 类型 | 说明 |
| :----: | :----: | :----: |  :----: |
| shopId | 是 | int |商家id |
| typeName | 是 | string |分组名称 |


 **返回参数说明** 
 
|参数名|类型|说明|
|:-----  |:-----|----- |
|data| int|更新数据库成功数量|
|msg|string|正确返回，这里为空。如果报错，这里是错误信息|
|ret|int|状态码 0:正常  -1:异常|


 **返回示例**
 ``` 
 {
	"data": 1,
	"msg": "",
	"ret": 0
}
```
