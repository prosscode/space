**简要描述：是否开启分销** 

**请求URL：** 
` http://localhost:9999/user/openDistribution?shopMark=A201902216&openFlag=1 `

**请求方式：**
- GET

**参数：** 

| 参数名 | 必选 | 类型 | 说明 |
| :----: | :----: | :----: |  :----: |
| shopMark | 是 | string |商家名称 |
| openFlag | 是 | int |0：默认为0，不开启；1：开启 |


 **返回参数说明** 
 
|参数名|类型|说明|
|:-----  |:-----|----- |
|data| int|更新数据库成功数量|
|msg|string|正确返回，这里为空。如果报错，这里是错误信息|
|ret|int|状态码 0:正常  -1:异常|


 **返回示例**
 ``` 
 {
	"data": 1,
	"msg": "",
	"ret": 0
}
```
