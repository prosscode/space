**简要描述：登录，暂时没有区分角色登录** 

**请求URL：** 
` http://localhost:9999/user/login?userName=pross&password=pross `

**请求方式：**
- GET

**参数：** 

| 参数名 | 必选 | 类型 | 说明 |
| :----: | :----: | :----: |  :----: |
| userName | 是 | string |用户姓名 |
| password | 是 | string |密码 |


 **返回参数说明** 
 
|参数名|类型|说明|
|:-----  |:-----|----- |
|data| int|更新数据库成功数量|
|msg|string|正确返回，这里为空。如果报错，这里是错误信息|
|ret|int|状态码 0:正常  -1:异常|


 **返回示例**
 ``` 
 {
	"data": 1,
	"msg": "",
	"ret": 0
}
```
