**简要描述：查询总金额/订单数** 

**请求URL：** 
` http://localhost:9999/order/totalPrice?shopId=6 `

**请求方式：**
- GET

**参数：** 

| 参数名 | 必选 | 类型 | 说明 |
| :----: | :----: | :----: |  :----: |
| shopId | 是 | int |商家id |


 **返回参数说明** 
 
|参数名|类型|说明|
|:-----  |:-----|----- |
|data| int|更新数据库成功数量|
|msg|string|正确返回，这里为空。如果报错，这里是错误信息|
|ret|int|状态码 0:正常  -1:异常|


 **返回示例**
 ``` 
{
    "data": [
        {
            "calPrice":124.97,
            "orderCount":2
        }
    ],
    "msg": "",
    "ret": 0,
    "count": 0
}
```
