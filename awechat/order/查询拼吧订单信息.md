**简要描述：查询拼吧订单** 

**请求URL：** 
` http://localhost:9999/wx/order/getSpellOrderInfo `

**请求方式：**
- GET

**参数：** 

| 参数名 | 必选 | 类型 | 说明 |
| :----: | :----: | :----: |  :----: |
| 可能必要加上用户所在位置的经纬度 | 是 | string | 方便计算距离 |


 **返回参数说明** 
 
|参数名|类型|说明|
|:-----  |:-----|----- |
|data| int|更新数据库成功数量|
|msg|string|正确返回，这里为空。如果报错，这里是错误信息|
|ret|int|状态码 0:正常  -1:异常|
