**简要描述：查询用户分销员身份，包括上级姓名** 

**请求URL：** 
` http://localhost:9999/wx/profile/getUserIdentity?userId=100 `

**请求方式：**
- GET

**参数：** 

| 参数名 | 必选 | 类型 | 说明 |
| :----: | :----: | :----: |  :----: |
| userId | 是 | int | 用户id |


 **返回参数说明** 
 
|参数名|类型|说明|
|:-----  |:-----|----- |
|data| int|更新数据库成功数量|
|msg|string|正确返回，这里为空。如果报错，这里是错误信息|
|ret|int|状态码 0:正常  -1:异常|


 **返回示例**
 ``` 
{
    "data": [
        {
            "id": 1,
            "shopId": 5,
            "userId": 100,
            "commission": null,
            "sumMoney": 190.5,
            "superUserId": 1,
            "superUserName": "pross",
            "nextUserNum": 12,
            "joinDate": "2019-03-23T08:23:26.000+0000",
            "isActive": 0
        }
    ],
    "msg": "",
    "ret": 0,
    "count": 0
}
```
